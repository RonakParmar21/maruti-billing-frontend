maruti billing frontend part

## Folder Structure 
- src 
    - assets (folder)
        - images (folder) (all images here)
        - icons (folder) (all icons here)
    - components (folder)
        - common (folder) 
            - all common component here 
            - Button.jsx (file)
            - Input.jsx (file)
            - Modal.jsx (file)
            - Loader.jsx (file)
            - etc...
        - layout (folder)
            - all layout component here
            - Navbar.jsx (file)
            - Footer.jsx (file)
            - Sidebar.jsx (file)
            - AdminSidebar.jsx (file)
    - pages
        - clinet (folder)
            - all client side pages here
            - Home.jsx (file)
            - Login.jsx (file)
            - etc...
        - admin (folder)
            - all admin side pages here
            - AdminLogin.jsx (file)
            - Dashboard.jsx (file)
            - etc...
    - routes (folder)
        - all routing files here
        - Approutes (file)
        - AdminRoutes (file)
        - ProjectRoutes (file)
    - services (folder)
        - backend connect here (base axios config)
        - api.js (file)
        - authService.js (file)
        - userService.js (file)
        - productService.js (file)
    - context (folder)
        - all context file here 
        - AuthContext.jsx (file)
        - ThemeContext.jsx (file)
    - hooks (folder)
        - all built-in and custom hooks here
        - useAuth.js (file)
    - utils (folder)
        - all utility files here 
        - contact.js (file)
        - helper.js (file)
    - app.jsx (file)
    - main.jsx (file)
    - index.css (file)
